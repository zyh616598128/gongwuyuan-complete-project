# Gongwuyuan Vue Frontend

现代化的公务员备考平台前端应用，基于 Vue 3、TypeScript、Vue Router 和 Pinia 构建。

## 特性

- Vue 3 with Composition API
- TypeScript 支持
- Vue Router 用于路由管理
- Pinia 用于状态管理
- 响应式设计
- 与后端 API 集成
- 模块化的代码结构

## 环境要求

- Node.js 16.x 或更高版本
- npm 或 yarn

## 安装和运行

1. 安装依赖：
   ```bash
   npm install
   ```
2. 启动开发服务器：
   ```bash
   npm run dev
   ```
