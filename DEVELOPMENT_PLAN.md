## 技术栈总结

### 后端技术栈
- **框架**: Spring Boot 2.7.0
- **持久层**: Spring Data JPA, Hibernate
- **数据库**: MySQL
- **缓存**: Redis
- **安全**: Spring Security, JWT
- **构建工具**: Maven
- **API文档**: Swagger/OpenAPI (待添加)

### 前端技术栈
- **开发语言**: Java
- **网络请求**: Retrofit, OkHttp
- **JSON解析**: Gson
- **图片加载**: Glide
- **UI框架**: Material Design Components
- **构建工具**: Gradle
